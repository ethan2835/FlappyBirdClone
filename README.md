# Flappy Bird Clone

A classic Flappy Bird clone built in Unity, following the step-by-step tutorial series. 

## 🎥 Tutorial Reference
* **Tutorial Series:** [CodeMonkey Flappy Bird Tutorial](https://www.youtube.com/watch?v=XtQMytORBmM&list=PL2t8PUpMvBy8lIJ-Nz6a78TGdzPSYgn5-&index=3)

## 🛠️ Built With
* **Engine:** Unity
* **Language:** C#

## 🐛 Known Issues & Bugs
* **First Pipe Score Bug:** The scoring system currently fails to register or count the very first pipe spawned at the start of a session. *(Work in progress to fix!)*
